# Pwc_aiagent

2026 삼일회계법인(Samil PwC) 하계인턴 AI Challenge 저장소. 삼일 업무에 도움이 되는 AI Agent 제작(개인과제)과 AI 기반 신규 서비스 기획(조별과제)을 진행한다.

## 저장소 구조

| 폴더 | 내용 |
|---|---|
| [개인과제/](개인과제/) | Extra Credit: 신입으로서 삼일 생활에 도움이 될 나만의 AI Agent 제작 |
| [조별과제/](조별과제/) | 선정 기업 니즈 발굴 → AI 기반 신규 서비스 기획. 현재는 7/16 팀 공유 전 개인별 사전 니즈발굴 단계 |

## 챌린지 개요

- 행사: 2026 하계인턴 Firmwide Training Week 중 AI Challenge (주관: L&E, 2026년 7월)
- 주제: "AI 시대, 삼일회계법인의 Next Service를 설계하다"
- 조별 과제: 우리 조가 선정한 기업의 니즈를 발굴하여 삼일에서 제공 가능한 AI 기반 신규 서비스를 기획
- 기간: 7월 16일(목)〜7월 29일(수), 조 편성 5〜6인
- 분량: PPT 표지·목차 제외 5장 이상 (발표 5분 분량)

## 일정

| 시점 | 내용 |
|---|---|

## 평가 기준

| 평가 항목 | 핵심 질문 |
|---|---|
| 문제 정의 | 문제와 원인이 명확하게 정의되었으며, 실제 기업의 pain point를 반영하고 있는가? |
| AI 적합성 | 해당 문제에 AI 활용이 필수적이며, 기술적·운영적 측면에서 실현 가능한가? |
| 고객 가치 | 삼일회계법인의 회계·세무·컨설팅 전문성을 바탕으로 고객에게 차별화된 가치를 제공할 수 있는가? |
| 기대효과 | 기대효과가 객관적 근거를 바탕으로 합리적으로 제시되었는가? |
| 전달력 | 문제 정의, 해결방안 및 기대효과가 논리적으로 연결되어 명확하게 전달되는가? |
| 가산점 (Agent/App) | 차별화된 사용자 경험 또는 업무 자동화를 효과적으로 구현하였는가? |
