# 덱 생성 프롬프트 (클로드 디자인용, 영어 프롬프트·한국어 산출물)

사용법: 아래 프롬프트를 통째로 전달한다. 대상 도구가 GitHub에 접근하지 못하면 [DATA SOURCES] 문단을 지우고 그 자리에 01〜06 md 내용을 직접 붙여넣는다. PPT를 네이티브로 내보내는 도구면 [OUTPUT FORMAT] 문단만 해당 도구의 내보내기 방식으로 교체.

```
[ROLE] You are an investment banking analyst and presentation designer. Build a concise
Information Memorandum slide deck for a fundraising pitch on KRAFTON (KRX 259960).
Audience: VC/PE investment professionals. Presentation length: 10 minutes.

[DATA SOURCES — read ALL of these before you start]
The folder "0714 조별과제" in the repository https://github.com/paranoiawd/Pwc_aiagent
is your ONLY data source.
- 01_재무데이터.md: annual/quarterly results, revenue mix, balance sheet, shareholders, shareholder returns
- 02_시장규모_TAM.md: global/India market size; Section 7 contains the TAM and target-segment logic
- 03_경쟁구도.md: domestic/global peer comparison, valuation, PUBG vs competing titles
- 04_하이라이트_소재.md: detailed evidence per highlight
- 05_Investment_Highlights_선정.md: the FOUR confirmed Investment Highlights (action-title wording,
  evidence, visualization suggestions, "Why now", counterargument rebuttals). Implement this document's
  logic and wording as-is. Do NOT re-select or invent highlights.
- 06_리스크_예상질문.md: risk responses. Only items 1 (single-IP dependence), 5 (India regulation),
  and 6 (Tencent) go on slides.
The PDFs in the folder (annual report amendments, etc.) are for cross-checking only, when a number
in the md files looks inconsistent.

[NUMBER USAGE RULES — highest priority, never violate]
- Use ONLY figures that appear in the md documents above. Do not supplement from web search or
  from your training memory.
- If a needed figure is not in the documents, drop that claim or mark it "미확인". Never fabricate.
- Never use items the documents flag as "미확인" or "사용 금지" (e.g., the Ariana Grande collab,
  the 16.14% stake figure for 장병규).
- Every figure gets a footnote at the bottom of the slide citing the ORIGINAL source and date as
  referenced inside the md files (e.g., "출처: 크래프톤 사업보고서 FY2025(2026.5 정정)",
  "Newzoo, 2026.3"). Do not cite md file names in footnotes.

[LANGUAGE] All slide content (action titles, body, footnotes, speaker notes) must be written in
Korean. Section labels may be in English (e.g., "Company Overview", "Key Financials",
"Investment Highlights", "Risk & Mitigants"). Do not write body text in English.

[OUTPUT FORMAT] 1920×1080 slides in a single self-contained HTML file. Include print CSS
(@page sized 1920×1080 landscape, one slide per page) so that printing from a browser
produces a clean PDF.

[STRUCTURE — Company/Industry/Business/Key Financials get exactly 1 slide each;
Highlights get 1 summary slide + at least 1 backup slide per highlight]
1   Cover: "KRAFTON" text wordmark, "Information Memorandum", "2026. 7.", "Strictly Confidential"
2   Company Overview (1 slide): company snapshot table (founding, listing, market cap, shareholders,
    shares outstanding), 3 key milestones, 4 KPI cards
3   Industry Overview (1 slide): TAM $201.6B plus the 3 target segments (use Section 7 logic of
    02_시장규모_TAM.md), competitive landscape table (Section 1 of 03_경쟁구도.md)
4   Business Overview (1 slide): revenue mix charts by platform and by region (overseas 91.6%),
    business model diagram (IP → multi-platform → live service)
5   Key Financials (1 slide): FY2021-FY2025 revenue bars + operating-margin line combo chart,
    1Q26 momentum (revenue +56.9%), summary table. Footnote that the 4Q25 operating-profit drop
    was one-off costs
6   Investment Highlights summary (1 slide): all FOUR highlights from 05_Investment_Highlights_선정.md
    on one slide — one action-title sentence + one representative figure each. This slide is the
    core message of the deck
7-10 Highlight backup slides (1 per highlight): implement each highlight from the 05 document.
    Allocate the most effort and polish here. Every backup slide has these 4 fixed elements:
    (a) action title (use the wording from 05)  (b) 2-3 pieces of supporting data
    (c) 1 visualization (follow the suggestion in 05)
    (d) bottom strip: one "Why now (catalyst)" line + one expected counterargument with rebuttal (from 05)
11  Risk & Mitigants + closing (1 slide): risks 1, 5, 6 from 06_리스크_예상질문.md with mitigation
    logic, plus one valuation reference line (12M forward P/E 10.6x vs peers 13-16x, from 03 document)

[DESIGN]
- Palette: navy #0A2540 (backgrounds/titles), white, and ONE accent color amber #F5A623 reserved
  for key figures only. No other colors
- Font: Pretendard, falling back to 'Malgun Gothic', system-ui. Body text minimum 28px,
  KPI numbers 60px or larger; footnotes are the only exception at 14-16px
- Every slide title is an action title (a complete sentence containing a key figure), 34-40px.
  Section labels in English, titles and body in Korean
- One core message per slide, max 4 bullets, no paragraph prose. Prefer charts, tables, diagrams
- Charts: no 3D, no gradients; label data points directly; consistent units (조원/억원);
  always show YoY growth alongside
- No external images or logo files (copyright + loading). Text and CSS shapes only
- Footer on every slide: page number + "Strictly Confidential"

[SELF-CHECK — run in order after finishing, fix, then deliver]
1. Search for "[", "XX", "TBD", "예:", "placeholder" — remove any leftovers
2. Cross-check every number in the deck against the 01-06 md files — delete any unsourced number
3. Verify every slide title is a complete sentence containing a figure
4. Verify no text below 28px except footnotes

[ADDITIONAL DELIVERABLE] Append speaker notes as text after the deck: ~30 seconds per slide,
10 minutes total, written in Korean. For each highlight, include a one-line rebuttal summary
of the expected counterargument, taken from 06_리스크_예상질문.md.
```
