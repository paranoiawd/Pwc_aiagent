# 다운로드 필요 원자료 URL 목록

이 환경의 프록시가 dart.fss.or.kr, krafton.com, kocca.kr, newzoo.com, lumikai.com, miraeasset.com, investgame.net 접근을 403으로 차단해 파일을 직접 받지 못했다. 아래 URL을 브라우저로 열어 수동 다운로드 후, `0714 조별과제/원자료/` 아래 권장 경로대로 넣으면 된다.

- 접근 상태 표기: "차단" = 이 환경에서 403 확인, "미검증" = 접근 시도 불가였으나 검색 결과에 실제로 나타난 주소. 지어낸 URL은 없음.
- DART 뷰어는 접속 후 상단 다운로드 버튼으로 원문 PDF 저장 가능.
