# 한국어 문체 지침 (어색한 한국어 교정)

한국어 문서·초안의 문장을 쓰거나 고치기 직전에 읽고 따른다. 내용·구조·포맷은 한국어_보고서_지침.md, 문장은 이 문서가 기준이다. 마치면 §11 교정 절차를 실행한다.

## 1. 영어 번역투

- 〜을 통해 → 〜로, 〜해서. "AI 기술을 통해 효율을 높인다" → "AI로 효율을 높인다"
- 〜에 의해 〜되다 → 행위자를 주어로. "정책에 의해 결정된다" → "정책이 결정한다"
- 〜을 가지다 → 있다, 〜다. "중요성을 가진다" → "중요하다"
- 〜중 하나다 → 〜다. "가장 큰 문제 중 하나다" → "큰 문제다"
- 아무리 강조해도 지나치지 않다 → 중요하다, 꼭 〜해야 한다
- 무생물 주어+타동사 → 사람·기관 주어. "보고서가 주장한다" → "보고서에 〜라고 적었다"
- "단순한 X가 아니라 Y" 구조(not just A but B) → 말하려는 사실만 쓴다
- 대명사(그, 그녀, 그것) 연발 → 이름·직함을 반복한다

## 2. 일본어투

- 〜에 있어서 → 〜에서, 조사 하나. "운영에 있어서의 문제" → "운영 문제"
- 〜에 다름 아니다 → 〜일 뿐이다, 〜와 같다
- 〜에의, 〜로의, 〜에서의, 〜으로서의 → 문장을 다시 짠다. "시장에의 진입" → "시장 진입"
- 조사 '의' 연쇄 → 빼거나 동사로 푼다. "부서의 데이터의 검증" → "부서 데이터 검증"
- 〜적(的) 남발 → 지워도 뜻이 같으면 지운다
- 〜하지 않으면 안 된다 → 〜해야 한다
- 〜에 지나지 않는다 → 〜일 뿐이다
- 상태·사실에 붙는 '〜고 있다' → 현재형. "갱신되고 있다" → "갱신된다" (지금 진행 중인 동작에만 쓴다)

## 3. 명사병 (명사 나열·압축)

- 서술성 명사+이다 문장은 동사로 푼다. "데이터 일치·사유 확인이 병목이다" → "데이터를 맞추고 사유를 확인하는 데 시간이 가장 오래 걸린다". "이견 조정이 반복 수요다" → "이견 조정 요청이 되풀이된다"
- 명사+명사+명사 사슬을 만들지 않는다. "변경 신호의 비정형 경로 전파" → "변경 신호가 비정형 경로로 퍼진다"
- 명사형 종결(〜임·함·음)과 명사구 압축은 개조식 문서·표 안에서만 쓴다. 산문에 섞지 않는다
- 사전에 없는 압축 조어를 만들지 않는다. 굳어진 실무 용어(재산정, 선반영 등)는 예외

## 4. 피동·완곡

- 이중 피동 금지: 되어지다, 보여지다, 쓰여지다 → 된다, 보인다, 쓰인다
- 피동 연발 금지: 한 문단에 '〜된다·〜진다'가 이어지면 행위자를 찾아 능동으로 바꾼다
- 다중 완곡 금지: "〜할 수 있을 것으로 보인다" → "〜할 것이다" 또는 "〜할지는 미확인이다". 불확실성 표현은 한 겹만
- "〜것이다", "〜할 수 있다" 종결이 반복되면 문장을 다시 짠다

## 5. 형식명사·군더더기

- 것·점·바·수·부분·과정·경우 남발 금지. "확인하는 과정이 필요하다" → "확인해야 한다". "〜하는 것이 중요하다" → "〜해야 한다"
- 〜에 대한, 〜와 관련하여, 〜의 경우 → 조사 하나로. "예산에 대한 논의" → "예산 논의"
- 동의어 중복 금지: 먼저 선행, 미리 예상, 서로 상충
- 강조 부사(매우, 정말, 상당히, 다양한) 절제: 지워도 뜻이 같으면 지운다

## 6. 접속·지시어

- 문두 접속사(또한, 이처럼, 따라서, 이를 통해, 이러한, 마침내, 나아가) 연발 금지. 논리가 문장 순서로 드러나면 접속사를 지운다
- 지시어(이러한, 해당, 동, 위) 남발 금지. 명사를 반복하는 쪽이 낫다
- 첫째·둘째·셋째 기계적 나열 금지. 항목이 진짜 병렬일 때만 번호를 쓴다

## 7. 참조·괄호

- 한 문장에 괄호는 1개까지. 넘으면 문장을 나누거나 본문에 푼다
- 괄호 참조가 이어지면 본문 주어로 바꾼다. "〜다(인터뷰 1·3)" 연발 → "경영기획 응답자 둘 다 〜라고 답했다"
- 같은 문장 안에서 말로 출처를 밝혔으면 괄호 참조를 겹쳐 달지 않는다
- 괄호 안에 문장급 정보를 넣지 않는다. 중요하면 본문 문장으로 꺼낸다

## 8. 메타 문장

- 발화·자료를 다시 설명하는 겹문장을 피한다. "이 발화가 담고 있는 사실은 〜라는 것까지다" → "이 말은 〜을 보여줄 뿐이다"
- 문서의 작성·인용 방침 서술 금지(보고서 지침 §3.8과 같다)

## 9. 물리 은유 남발 (AI 습관)

- 추상 개념에 촉각·물리 동사를 붙이는 습관을 경계한다: 새다, 눌리다, 붙다, 얹히다, 닿다, 흔들다, 심어지다, 파고들다, 단단하다, 무르다
- 영어 은유의 직역이 대부분이다. "신호가 회계에 닿는다" → "신호가 회계에 반영된다". "EAC를 흔드는 신호" → "EAC를 바꿀 신호". "일정이 눌린다" → "일정이 밀린다". "논거가 단단하다" → "근거가 충분하다"
- 판별법: 한국어 화자가 실제로 쓰는 조합인지 따진다. 굳어진 관용구(문제가 터지다, 일이 몰리다, 왜곡이 쌓이다, 예산이 새다)는 자연스럽고, 직역 은유는 어색하다

## 10. 리듬·낭독

- 문장 길이와 종결어미가 균일하면 고친다(〜다·〜다·〜다 연속)
- 쉼표로 절을 계속 잇지 않는다. 쉼표 2개가 넘으면 문장을 나눈다
- 최종 점검은 소리 내어 읽기다. 읽다가 걸리는 문장은 어딘가 잘못됐다

## 11. 교정 절차 (출고 전)

1. 어미 훑기: 종결어미 3연속, 피동 연속, "것이다·수 있다" 반복
2. 패턴 검색: 통해, 있어서, 에 의해, 에 대한, 되어, 고 있다, 것, 중 하나
3. 괄호 세기: 문장당 1개 초과, 겹참조, 괄호 속 문장급 정보
4. 명사+이다 서술 찾기: 동사로 풀 수 있으면 푼다
5. 물리 은유 찾기: 추상 개념에 붙은 촉각·물리 동사를 관용구인지 직역인지 따진다
6. 소리 내어 읽기

교정 철칙: 사실·수치·고유명사는 그대로 둔다. 걸린 곳만 고치고 멀쩡한 문장을 갈아엎지 않는다. 규칙과 자연스러움이 충돌하면 자연스러움이 이긴다.
