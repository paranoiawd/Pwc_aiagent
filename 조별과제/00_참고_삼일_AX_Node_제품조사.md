# 삼일PwC AX Node 제품 조사 (2026-07-08)

조별과제 To-Be(신규 AI 서비스) 설계 시 삼일의 기존 자산과 겹치는지, 무엇을 재사용할 수 있는지 확인하기 위한 조사. pwc.com과 일부 언론이 봇 차단이라 검색 색인 기반으로 확인했고, 원문 직접 대조가 안 된 항목은 표기함.

## 1. AX Node 개요

- 2025년 7월 공식 출범한 AI·Digital 혁신 전담조직. 명칭은 기술과 데이터를 모아 확장하는 연결점(Node)이라는 의미
- 규모: 약 90명, 3년간 약 200억 원 투자 계획, FY2025 디지털 솔루션 매출 약 200억 원, 자체 솔루션으로 연간 약 20만 시간 절감 (전자신문 2025.12.5 단일 출처)
- 서비스 라인 4개: Digital & AI Transformation 컨설팅, AI Solutions, ERP 도입·전환 자문, 통합경영지원(BPO)
- 사업 구조: 감사·세무·내부통제 등 내부 업무를 먼저 AI화하고, 그 결과물을 일반 기업 대상 구독 제품으로 외판하는 순서

## 2. 제품별 상세

| 제품 | 한 줄 정의 | 상태 |
|---|---|---|
| Tax Agent | 세무 질의응답 생성형 AI 에이전트 | 2025.10 정식 출시, 구독형 |
| AI Accountant | 회계·세무 특화 LLM 질의응답 | 내부 실무 활용 + 대외 서비스 |
| K-SOX Hub | 내부회계관리제도 전 과정 AI 솔루션 | 국내 서비스 중, 2026.4 해외 수출 |
| Chat X | 기업 보안 환경용 사내 생성형 AI 플랫폼 | SaaS·온프레미스 판매 |
| Accounting Insight | 회계기준·사례 대화형 챗봇 | 서비스 중 |
| Robotic Platform | 공시(재무제표·주석) 자동화 | 서비스 중 |
| Easy View | 엑셀 기반 분석·자동화 대시보드 | 서비스 중 |

### Tax Agent (2025.10 정식 출시)

- 삼일아이닷컴이 54년간 축적한 세무 데이터(법령·예규·판례, 매주 업데이트)를 RAG로 결합한 세무 전문 생성형 AI. 자연어로 질문하면 근거 문서 기반 답변을 제공하고, 질문이 모호하면 AI가 필요한 정보를 역으로 요청함
- 자체 개발 검색 알고리즘이 허깅페이스 MTEB 리더보드 1위 기록(회사 발표 수치)
- MS Azure 프라이빗 LLM 구동, ISO 27001, 고객 데이터의 학습 사용 차단. 2주 무료 체험 제공
- 타깃: 기업 세무팀과 세무대리인. 즉 세무 담당 조직이 있는 규모의 회사

### AI Accountant

- ChatGPT 파운데이션 모델에 회계기준서·공시 규정·삼일 내부 지침 DB를 RAG로 결합한 회계·세무 특화 LLM
- 기업 회계처리 질의에 근거 문서 기반 추론 답변. 감사 실무에서 특수관계자 판별 같은 절차 지원에도 사용(조세일보·택스워치 2024.11 보도)

### K-SOX Hub / Control Testing Automation

- 내부회계관리제도(K-SOX)의 위험평가, 설계평가, 운영평가, 미비점 관리, 외부감사인 대응까지 전 과정을 다루는 엔드투엔드 AI 솔루션
- 핵심은 내부통제 운영 적정성의 AI 자동 평가. 수작업 반복 테스트를 자동화해 시간과 평가 편차를 줄임
- 2026.4.13 PwC 해외 회원사에 SaaS 형태 공급 발표. 국내 회계법인이 자체 개발한 AI 솔루션의 첫 해외 수출 사례. 공급 대상은 내부통제 수준이 높은 선진국이며 해외 일반 기업으로 확대 추진

### Chat X

- 기업 보안 환경에서 쓰는 ChatGPT형 사내 AI 플랫폼. 문서·이미지·표·PPT를 이해하고 생성하는 멀티모달 지원
- 여러 최신 LLM을 선택·조합하는 멀티모델 구조, 사내 문서·규정·ERP 연동 RAG 내장. 삼일 보안 하의 SaaS 또는 온프레미스 설치형

### Accounting Insight

- 삼일이 자체 구축한 회계 DB 기반 대화형 챗봇. 최신 회계기준과 사례에 근거한 질의응답 제공

### Robotic Platform (공시자동화)

- Smart Reviewer와 DSD Wizard로 재무제표·주석의 작성, 검토, 비교를 자동화. 정기보고서·공정위 공시·금융기업 특화 공시의 데이터 집계와 DSD 자동 작성 지원

### Easy View

- 기업이 실무에서 쓰는 엑셀 기반 분석·검증·자동화 도구를 고도화한 대시보드

### 협력·확장 (부수)

- Next Generation PA 연결 플랫폼: 온라인 플랫폼 정산 솔루션, 코리아포트원과 고도화 MOU
- 모티프테크놀로지스와 AI 에이전트 공급 MOU(2025.8), 마크앤컴퍼니와 스타트업 AX 프로그램 운영

## 3. 과금 방식 (확인된 것)

SaaS 구독형, 에이전트 사용량 기반 과금, 절감 효과·리스크 발견 금액에 연동한 성과 공유형을 병행 중(전자신문 2025.12.5). 성과 공유형 선례가 이미 있다는 점이 중요함.

## 4. 조별과제 시사점

- 전 제품이 세무팀·회계팀·내부회계 조직을 갖춘 상장사·중견 이상을 타깃으로 함. 관리 인력 2〜3명뿐인 중소 제조기업용 제품은 없음. 기획 중인 "중소 제조 AI 관리부"는 기존 라인업과 중복되지 않고 고객층을 아래로 확장하는 위치임
- 재사용 자산이 명확함: Tax Agent·AI Accountant의 세무·회계 RAG 엔진, Chat X의 멀티모달 문서 처리 플랫폼, K-SOX SaaS 운영 경험. "이미 있는 엔진을 견적서·성적서·거래명세서라는 중소 제조 문서에 붙인다"가 실현 가능성 논리가 됨
- 지불 여력 반박 대응: 성과 공유형 과금이 사내 선례로 존재하므로, 중소기업에는 절감액 연동 과금을 제안할 근거가 있음
- 수치 인용 주의: 90명·200억·20만 시간은 전자신문 단일 출처라 기획안 인용 시 "언론 보도 기준" 표기 필요
