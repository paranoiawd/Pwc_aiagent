## 확인된 사실
- 한화시스템 ICT부문의 제조&건설 시스템 통합(SI) 실적 페이지에 한화오션 관련 구축 사례가 게재되어 있음.
- 한화오션 데이터 거버넌스 마스터플랜(Master Plan) 수립: 전사 데이터 관리체계 수립부터 데이터 표준화·품질 관리 추진을 위한 실행 계획 확립. 디지털 전환의 기반 확보로 소개됨.
- 한화오션 블록 물류비용 및 적치장 최적화 사례: 최적화·데이터 분석으로 작업 스케줄링 수립시간 단축, 운송 효율 향상, 비용 절감 달성.
- 한화시스템 AI/Data Governance 서비스: 데이터 사일로·품질 저하 문제 해소를 위한 메타데이터 표준사전·운영 프로세스 정립, 솔루션 도입을 통한 데이터 거버넌스 시스템 구축·고도화.

## 미확인 사항
- 수집 지시문에 언급된 "선표계획 주요리소스 최적화 시스템 구축" 문구는 페이지 원문 접근 불가로 직접 확인하지 못함. 검색 결과에서는 데이터 거버넌스 마스터플랜과 블록 물류·적치장 최적화 사례만 확인됨. 원문 페이지에 해당 실적이 추가로 게재되어 있을 가능성은 있음.

## 반대근거로서의 의미
한화오션은 이미 그룹 계열 SI(한화시스템)를 통해 데이터 거버넌스 마스터플랜과 생산 리소스 최적화 시스템을 구축했거나 구축 중. "데이터 기반 계획·경보 체계가 부재하다"는 전제와, 외부 신규 AI 솔루션이 파고들 공백이 크다는 논리를 약화. 계열사 내재화 경로가 우선될 가능성도 시사.
