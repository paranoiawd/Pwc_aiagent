# 실무자 인터뷰 기록 [가상]

성격: 실제 인터뷰가 아니다. [인터뷰_준비.md](인터뷰_준비.md)의 예상 답변 가설과 [한국_중소제조업_이해.md](한국_중소제조업_이해.md)·[업종선정_리서치.md](업종선정_리서치.md)의 통계·업계 구조를 바탕으로 재구성한 가상 문답이며, 팀 공유 시에도 가상임을 밝힌다.

## 1. 개요

- 회사(가상): (주)동주정밀. 자동차부품 프레스·CNC 가공 2차 협력사, 시화공단 소재
- 규모(가상): 직원 38명(사무 5·생산 31·품질 2), E-9 외국인 9명, 연매출 약 120억, 1차 벤더 2개사가 매출의 약 80%
- 인터뷰이(가상 인물): 관리부 김 차장. 경리·총무·구매 겸직 12년 차
- 진행 방식: 리서치 기반 페르소나 문답(실제 일시 없음)

## 2. 공식 질문 문답

### 2-1. 가장 많은 시간을 투입하는 업무

- 월 마감 주간(말일〜익월 5일): 거래명세서 대사·세금계산서 발행·수금 확인에 이틀 이상 집중
- 평상시: 견적서 작성(도면 보고 재료비·공수 산출) 주 6〜8시간, 자재 발주·입고 관리 주 5시간
- 4대보험·급여·연차 관리 월 1〜2일

### 2-2. 반복적으로 수행하는 업무

- 발주서(PO) 접수 → 생산 지시 전달 → 생산일보·재고 엑셀 입력 → 거래명세서·성적서 발행이 매일 반복
- 같은 수주 정보를 엑셀 관리대장, 더존(회계), 원청 구매 포털에 세 번 입력
- 검사성적서는 품질 담당이 측정값을 수기로 옮겨 납품 로트마다 작성

### 2-3. 업무 중 가장 불편한 점

- 채용 불능: 생산직 공고 3개월째 지원자 0명, E-9 배정 대기 중
- 사양 변경·긴급 발주가 전화·카톡으로 와서 기록이 안 남음. 사후에 "말이 다르다" 분쟁
- 과거 자료 검색: 3년 전 유사 도면의 견적 단가를 찾는 데 반나절
- 중대재해법 확대 적용 이후 안전 점검 서류 증가, 월 1회 외부 컨설팅으로 대응

### 2-4. 반복적인 거래처 문의·불만

- 납기 확인 전화 하루 5〜6통("언제 나와요"), 진행 상황·재고 확인
- 성적서·거래명세서 재발행 요청 주 2〜3건, 스팟 견적 요청 주 3〜4건
- 응대가 관리부 본인과 공장장 두 사람에게 집중

### 2-5. 거래처 응대 과정의 어려움

- 긴급 발주·납기 단축을 거절 못 함(매출 80%가 2개사). 잔업·특근으로 흡수
- 연례 CR(단가 인하) 협상에서 방어할 원가 데이터가 없어 감으로 대응
- 납기 문의에 즉답 불가. 현장 확인 후 회신하느라 전화가 2〜3회 오감

### 2-6. 현재 AI를 활용 중인 업무

- 회사 차원 활용 없음. 본인이 챗GPT를 외국인 직원 공지문 번역에 써본 정도
- 2023년 정부지원으로 기초 MES 도입했으나 현장 입력이 안 돼 사실상 방치("지원금 받고 깔았는데 쓰는 사람이 없다")

### 2-7. AI가 가장 먼저 도입되길 바라는 부분 (객관식 보기 제시)

- 보기: ① 도면 견적 초안 ② 성적서·서류 자동 생성 ③ 거래처 문의 자동 응답 ④ 재고·발주 알림 ⑤ 작업지시 번역
- 1순위 ②: "명세서·성적서 만드는 시간만 없어져도 사람 반 명 몫"
- 2순위 ①: 견적은 공장장과 본인만 가능해 둘 다 부재 시 회신 지연
- ③은 반신반의(거래처가 기계 응답을 꺼릴 것), ⑤는 현장 반장이 필요하다는 입장

## 3. 확장 문답 발췌

- 주문의 일생(B-1): 도면·사양 접수 → 견적 → PO → 자재 발주 → 작업지시서(수기) → 가공·검사 → 성적서·거래명세서 → 납품 → 월 마감 세금계산서 → 약 60일 후 수금. 문서 7종 중 전산에서 자동 생성되는 것은 세금계산서뿐
- 사고 사례(B-2): 올해 2월 자재 입고 지연을 납기 이틀 전에 파악, 특근·외주로 막았으나 이익 잠식. 원인은 재고 엑셀과 실물 불일치 / 작년 견적에서 열처리 외주비 누락, 해당 로트 내내 적자 납품
- 사람·조직(B-3): 신입 교육은 어깨너머, 매뉴얼 없음. 퇴직한 전 공장장의 단가·거래처 이력이 인수인계되지 않음
- 돈·거래(B-4): 제품별 원가를 정확히 모름("대충 재료비에 얼마 곱하는 식"). 수금은 월 마감 후 60일 전자어음 위주
- 규제·서류(B-5): IATF 16949 유지 문서를 품질 1명이 전담, 심사 시즌 야근. 올해 원청이 탄소배출량 설문을 처음 요구했으나 작성법을 몰라 일부 빈칸 제출

## 4. Pain Point 후보 (누가·언제·얼마나 자주·얼마나 아픈지)

1. 문서 반복 노동: 관리부 2명이 견적서·거래명세서·성적서·세금계산서를 수작업 반복 작성하고 같은 정보를 3곳에 입력, 주 15시간 이상. 마감 주간 과부하 (가설 2 확인)
2. 데이터 실종: 생산·재고·과거 단가가 엑셀·수첩·특정인 머릿속에 분산. 납기 문의 하루 5〜6건에 즉답 불가, 견적 항목 누락으로 적자 납품 발생 (가설 3 확인)
3. 도입 실패 경험: 지원금으로 MES를 도입했으나 입력 부담으로 방치. 자금과 운영 인력 부족이 재도입 장벽 (가설 1 부분 확인)
