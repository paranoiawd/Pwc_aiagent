# 모의 VDR 교차검증 노트 (2026-07-08)

검증 기준: ① 삼일회계법인 중소벤처기업 M&A지원센터 『M&A Guide Book』 개정판(2025.11, 저장소 `개인과제/samilpwc_mna_guide-book.pdf`) 정독 ② VDR 업체(iDeals·FirmRoom 등)의 데이터룸 인덱스 표준 ③ 회계·세법 규정 재검산.

## 현실과 정합함이 확인된 것

- 폴더 구조: 표준 데이터룸 인덱스(회사일반/재무/세무/법무/영업/인사/자산/IT/규제/보험 10대 카테고리)와 일치
- 인덱스 파일·미수령 자료 추적: VDR 운영 관행(권한 관리, 업로드 추적)과 부합. 인덱스 관리가 주니어 업무라는 설정도 실제와 같음
- 발견 포인트 4종은 가이드북의 실사 방법론과 정확히 대응됨
  - 일회성 정부보조금 → QoE의 Normalization 조정(일회성 항목 배제) 사례
  - 특수관계자 임차료 → QoE Pro-forma 조정 + 세무실사(TDD)의 지배주주 거래 적정성·부당행위계산 검토 항목
  - 퇴직급여충당부채 과소 → QoA의 "부채성 항목"(가이드북이 퇴직급여충당부채를 명시적으로 지목)
  - 소송 충당 미계상 → 법률실사의 우발채무 중심 검토, 진술보장·공개목록 연계
- 채권 연령분석·재고 장기체화·매입채무 검토 포인트: 가이드북 QoA 순운전자본 체크리스트와 일치
- 딜 문서 흐름(Teaser→NDA→LOI→IM→MOU→SPA), 이행보증금, 독점협상권, 진술보장·공개목록 면책: 가이드북 Q7·Q46〜50과 일치. README에 반영
- 승계 부재형 창업주 매각 시나리오: 가이드북이 중소 M&A의 구조적 트렌드로 명시

## 발견·수정한 오류

1. 법인세 산출세액 계산 오류: 최초 작성본은 세율 구조(과표 2억 이하 9%, 2억 초과 19%)와 안 맞는 수치였음 → 세무조정(법인세비용 손금불산입 포함)부터 재구성해 과표 205/330/510, 산출세액 19/43/77로 수정. 손익 법인세비용과 결정세액의 차이(지방소득세·이연법인세) 주석 추가
2. 부가세 신고 주기 오기: "반기별 확정신고(예정고지)"로 썼으나 이 규모 법인은 분기별 예정신고 의무 → 수정
3. 회계기준 오기: 자산 145억·부채 82억·매출 183억이면 외감 4지표 중 3개 충족으로 외부감사 대상 → "중소기업회계기준" 표현을 일반기업회계기준으로 수정하고 감사보고서에 외감 근거 주석 추가
4. 감사인 명칭을 가상명(가온회계법인)으로 명확화

## 남은 한계 (현실과의 의도적 차이)

- 실제 VDR은 스캔 PDF·엑셀 원본이 수백〜수천 건이고 파일당 정보 밀도가 낮다. 본 모의 자료는 문서당 요약 밀도가 높은 md/csv라 "실물 재현"이 아니라 "문서 종류·내용·상호참조의 재현"이다. 에이전트 데모 목적에는 이 형태가 적합
- 원본 양식(등기부·세무신고서 서식 등)의 시각적 재현은 생략. 필요 시 데모용으로 일부 문서만 PDF 서식화 가능
- 금액·거래처·인명은 전부 가상이며 숫자 정합성은 문서 간 상호 일치 수준으로만 맞춤
