# 모의 데이터룸 (Project Gear)

본 폴더는 M&A 재무실사(FDD) 데이터룸을 재현한 학습·데모용 모의 자료다. 모든 문서는 가상 기업 "(주)한결정밀"에 대한 합성 데이터로, 실존 기업·인물·거래와 무관하다. 개인과제(데이터룸 에이전트)의 데모 코퍼스로 사용한다.

## 딜 시나리오

- 프로젝트명: Project Gear (딜은 관행상 코드네임으로 부른다)
- 대상회사: (주)한결정밀. 1998년 설립, 자동차 프레스·CNC 가공 부품 2차 협력사, 경기 화성 소재, 임직원 45명
- 거래 구조: 창업주(지분 80%)·가족(20%)의 구주 매각, 전략적 투자자가 인수 검토
- 재무 기준일: 2025년 12월 31일 (FY2025)
- 자문 구도: 삼일PwC가 매수인 측 재무실사(FDD) 수행 가정

## 심어둔 발견 포인트 (에이전트 데모용 정답지)

1. 특수관계자 거래: 대표이사 배우자 소유 창고를 시세 대비 고가로 임차 (07_자산부동산/임대차계약서_창고, 05_인사노무·01_회사일반과 교차 확인 가능)
2. 일회성 수익: FY2025 영업외수익에 스마트공장 정부보조금 120백만 원 포함 (02_재무회계/감사보고서, 03_세무와 대조)
3. 퇴직급여충당부채 과소계상 추정: 충당 설정률이 추계액 대비 낮음 (05_인사노무/퇴직급여충당_내역)
4. 진행 중 소송: 품질 클레임 손해배상 300백만 원, 충당부채 미계상 (06_법무/소송및분쟁현황)

## 폴더 구조

실제 VDR의 표준 인덱스(회사일반/재무/세무/영업/인사/법무/자산/IT/환경안전/보험)를 따른다. 전체 문서 목록은 [00_데이터룸_인덱스.md](00_데이터룸_인덱스.md) 참조. 인덱스 관리가 바로 신입이 하는 일이다.
