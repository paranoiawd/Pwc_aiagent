# 모의 데이터룸 (Project Gear)

본 폴더는 M&A 재무실사(FDD) 데이터룸을 재현한 학습·데모용 모의 자료다. 모든 문서는 가상 기업 "(주)한결정밀"에 대한 합성 데이터로, 실존 기업·인물·거래와 무관하다. 개인과제(데이터룸 에이전트)의 데모 코퍼스로 사용한다.

## 딜 시나리오

- 프로젝트명: Project Gear (딜은 관행상 코드네임으로 부른다)
- 대상회사: (주)한결정밀. 1998년 설립, 자동차 프레스·CNC 가공 부품 2차 협력사, 경기 화성 소재, 임직원 45명
- 거래 구조: 창업주(지분 80%)·가족(20%)의 구주 매각(승계 부재형 매각, 중소 M&A의 전형), 전략적 투자자가 인수 검토. 개별 협상(Private Deal) 방식
- 진행 단계: Teaser → NDA 체결 → IM 수령 → LOI 제출 → MOU(독점협상권) → 현재 본실사 단계. 본 VDR은 우선협상대상자 선정 후 개설된 것
- 재무 기준일: 2025년 12월 31일 (FY2025). 가격조정은 Locked Box 방식 협의 중
- 자문 구도: 삼일PwC가 매수인 측 재무실사(FDD) 수행 가정
- 기업결합신고: 대상회사 자산·매출이 300억 원 미만이라 공정거래법상 신고 대상 아님

## 현실 관행 메모 (삼일 M&A 가이드북 교차확인)

- VDR은 전문 업체 플랫폼에서 운영되며 등록 인원별 접속 권한 부여, 열람·다운로드 로그 추적, 권한 차단 시 기다운로드 자료도 열람 불가
- 보안 수준이 높은 원본(계약서 원본·의사록 원본·영업기밀)은 VDR에 올리지 않고 PDR(현장 열람)로 제공하는 관행이 있음. 본 모의 VDR의 "요약 전사"·"마스킹"·"현장 열람" 표기는 이를 반영
- 매도자가 데이터룸에 공개한 사항은 SPA 진술보장 위반 손해배상에서 면책되는 방향으로 협상되는 것이 일반적(공개목록 효과). 즉 데이터룸은 매수인의 정보 획득 수단이자 매도인의 책임 방어 수단

## 심어둔 발견 포인트 (에이전트 데모용 정답지)

1. 특수관계자 거래: 대표이사 배우자 소유 창고를 시세 대비 고가로 임차 (07_자산부동산/임대차계약서_창고, 05_인사노무·01_회사일반과 교차 확인 가능)
2. 일회성 수익: FY2025 영업외수익에 스마트공장 정부보조금 120백만 원 포함 (02_재무회계/감사보고서, 03_세무와 대조)
3. 퇴직급여충당부채 과소계상 추정: 충당 설정률이 추계액 대비 낮음 (05_인사노무/퇴직급여충당_내역)
4. 진행 중 소송: 품질 클레임 손해배상 300백만 원, 충당부채 미계상 (06_법무/소송및분쟁현황)
